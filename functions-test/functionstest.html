Question 1:

- In this excersise, we are building the largest number finder function:

  1. You should create a function named largestOfAll
  2. largestOfAll should take a single parameter.
  3. largestOfAll([[4, 5, 1, 3], [13, 27, 18, 26], [32, 35, 37, 39], [1000, 1001, 857, 1]]) should return an array.
  4. largestOfAll([[13, 27, 18, 26], [4, 5, 1, 3], [32, 35, 37, 39], [1000, 1001, 857, 1]]) should return [27, 5, 39, 1001]
  5. largestOfAll([[4, 9, 1, 3], [13, 35, 18, 26], [32, 35, 97, 39], [1000000, 1001, 857, 1]]) should return [9, 35, 97, 1000000]
  6. largestOfAll([[17, 23, 25, 12], [25, 7, 34, 48], [4, -10, 18, 21], [-72, -3, -17, -10]]) should return [25, 48, 21, -3]

- Thinking Strategy → Pick a starting point → say “the first element is the biggest I know.” (like setting a “champion” at the beginning)

  1. Go through the rest one by one. Each time:
  2. Compare the current number with the “champion.”
  3. If the current number is bigger → replace the champion.
  4. If not → keep the champion.
  5. After the loop ends → the champion is the largest number.
  1. Start with an empty result array (this will hold all the champions).
    1. For each sub-array:
    2. Assume the first number is the champion.
    3. Loop through the rest of that sub-array.
    4. Update the champion if a bigger number is found.
    5. After finishing that sub-array, push the champion into the result array.
  2. After checking all sub-arrays, return the result array.

Answer 1:

function largestOfAll(arrays){
let result = [];
for(let i = 0; i < arrays.length; i++){
let champion = arrays[i][0];
for(let number of arrays[i]){
if(number > champion){
champion = number;
}
}
result.push(champion);
}
console.log(result);
}
